## üéØ Quick Start

### AI Course Setup
```bash
python -m venv ai_course_env
source ai_course_env/bin/activate
pip install -r requirements.txt
```

### SaaS Platform Setup
```bash
git clone <repository-url>
cd saas-marketing-platform
pip install -r requirements.txt
npm install
python manage.py migrate
python manage.py runserver
```
