## 🚀 Quick Start

### 1. Installation
```bash
git clone https://github.com/yourcompany/referral-contest-system
cd referral-contest-system
npm install
```

### 2. Configuration
```bash
cp .env.example .env
# Add your SendGrid API key and database credentials
```

### 3. Run Your First Campaign
```bash
npm run start
node scripts/send-test-campaign.js
```
